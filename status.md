## API说明 ##

* 原型定义

~~~{.c}
int function(int a, int b);
~~~
* 说明
 * 函数就是函数。
* 参数
 * a, 参数a指示出是balabala
 * b, 参数b指示出是balabala
* 返回值
 * 返回0，代表；
 * 返回1，代表很好。
* 例子

~~~{.c}
  void test(void)
  {
    function(10, 100);
  }
~~~

这个例子说明了某某用法，在第1行balabala，第2行balabala。最终的输出结果类似：

  ~~~{.txt}
  balabala
  ~~~
